# 퍼셉트론 다음 개념

퍼셉트론을 이해했다면 다음으로 봐야 할 핵심 요소는 아래와 같습니다.

- 가중치
- 편향
- 활성화 함수
- 학습 과정

이 요소들이 합쳐져 실제 신경망의 동작이 구성됩니다.

## 1. 가중치

가중치는 각 입력값의 중요도를 나타내는 값입니다.

입력이 여러 개 들어올 때 모든 입력을 똑같이 취급하지 않고, 어떤 입력은 더 크게, 어떤 입력은 더 작게 반영하기 위해 사용합니다.

### 예시

```text
입력: 공부시간, 수면시간, 출석률
출력: 시험 합격 여부
```

이 경우 모델은 각 입력을 다음처럼 다르게 볼 수 있습니다.

```text
공부시간  -> 중요도 큼
수면시간  -> 적당한 영향
출석률    -> 추가적인 영향
```

즉, 가중치는 입력별 영향력을 조절하는 역할을 합니다.

### 수식

```text
x1w1 + x2w2 + x3w3
```

- `x`: 입력값
- `w`: 가중치

입력값에 가중치를 곱한 뒤 모두 더해서 다음 단계로 전달합니다.

## 2. 편향

편향은 출력이 결정되는 기준점을 조정하는 값입니다.

가중치만 있으면 모든 판단 기준이 원점을 중심으로 정해지는데, 편향을 추가하면 그 기준을 위아래 또는 좌우로 이동할 수 있습니다.

### 수식 형태

```text
x1w1 + x2w2 + x3w3 + b
```

- `b`: 편향

쉽게 보면 편향은 기본값을 조정하는 역할입니다.

### 예시

시험 합격 예측 모델이 있다고 가정해보면,

```text
공부시간 × 가중치 + 출석률 × 가중치
```

만으로 판단하면 기준이 너무 딱딱할 수 있습니다.

여기에 편향을 더하면 다음과 같은 조정이 가능합니다.

```text
기본적으로 조금 더 합격 쪽으로 보정
또는
기본적으로 조금 더 불합격 쪽으로 보정
```

즉, 편향은 판단 기준을 이동시키는 역할을 합니다.

## 3. 활성화 함수

입력값과 가중치, 편향을 모두 계산한 결과를 바로 출력하지 않고 한 번 더 가공하는 함수가 활성화 함수입니다.

### 전체 흐름

```text
입력값
-> 가중치 곱셈
-> 모두 더함
-> 편향 추가
-> 활성화 함수 적용
-> 최종 출력
```

### 활성화 함수가 필요한 이유

가중치와 편향만으로는 단순한 선형 계산만 가능합니다.
이 상태로는 복잡한 패턴을 학습하는 데 한계가 있습니다.

활성화 함수를 넣으면 비선형성을 추가할 수 있고, 직선 하나로 나눌 수 없는 문제도 다룰 수 있게 됩니다.
이 점이 신경망 성능을 크게 좌우하는 핵심 요소입니다.

### 대표적인 활성화 함수

#### 1. Step Function

퍼셉트론 초기 형태에서 사용된 방식입니다.
기준값을 넘으면 `1`, 아니면 `0`을 출력합니다.

```text
값이 0 이상 -> 1
값이 0 미만 -> 0
```

장점

- 단순한 구조
- 이진 분류에 직관적

단점

- 미분이 어려워 학습에 불리함

#### 2. Sigmoid

출력을 `0~1` 사이로 변환하는 함수입니다.

```text
큰 음수 -> 0에 가까움
0 부근 -> 0.5
큰 양수 -> 1에 가까움
```

장점

- 확률처럼 해석 가능
- 출력 범위가 명확함

단점

- 깊은 신경망에서 기울기 소실 문제가 발생할 수 있음

#### 3. ReLU

현재 가장 널리 쓰이는 활성화 함수 중 하나입니다.

```text
음수 -> 0
양수 -> 그대로 출력
```

장점

- 계산이 단순함
- 학습이 빠름
- 깊은 신경망에서 많이 사용됨

단점

- 음수 구간에서는 값이 계속 `0`이 될 수 있음

## 4. 학습 과정

신경망의 학습이란 정답에 가까운 출력을 내도록 가중치와 편향을 조정하는 과정입니다.

즉, 처음에는 가중치와 편향이 적절하지 않지만 데이터를 반복해서 보면서 점점 더 좋은 값으로 바뀌는 구조입니다.

### 기본 흐름

```text
1. 입력 데이터 넣기
2. 현재 가중치로 출력 계산
3. 정답과 비교
4. 오차 계산
5. 오차를 줄이도록 가중치와 편향 수정
6. 반복
```

이 반복 과정이 학습의 핵심입니다.

## 5. 손실 함수

손실 함수는 모델의 예측이 정답과 얼마나 차이 나는지 수치로 나타내는 함수입니다.
이 값을 기준으로 모델이 얼마나 틀렸는지 판단합니다.

### 예시

- 집값 예측이 많이 틀리면 손실이 큼
- 스팸 분류가 틀리면 손실이 증가함
- 정답과 거의 같으면 손실이 작음

즉, 손실 함수는 모델의 오답 정도를 측정하는 역할을 합니다.

## 6. 역전파

역전파는 오차를 줄이기 위해 각 가중치를 얼마나 수정해야 하는지 계산하는 과정입니다.

출력층에서 발생한 오차를 뒤쪽에서 앞쪽으로 전달하면서, 각 층의 가중치가 결과에 얼마나 영향을 주었는지 계산합니다.
이 과정을 통해 어느 가중치를 얼마나 바꿔야 할지 결정할 수 있습니다.

### 핵심 의미

```text
결과가 틀렸다면
어떤 가중치가 그 오류에 얼마나 기여했는지 계산
-> 그만큼 수정
```

이 과정이 역전파입니다.

## 7. 정리된 전체 흐름

신경망 전체 동작은 다음처럼 정리할 수 있습니다.

```text
입력값
-> 가중치 반영
-> 편향 추가
-> 활성화 함수 적용
-> 출력 생성
-> 정답과 비교
-> 손실 계산
-> 역전파로 수정량 계산
-> 가중치와 편향 업데이트
-> 반복 학습
```

## 핵심 정리

- 가중치: 입력별 중요도
- 편향: 판단 기준 조정값
- 활성화 함수: 비선형성 부여
- 손실 함수: 틀린 정도 측정
- 역전파: 수정 방향 계산
- 학습: 가중치와 편향을 반복적으로 조정하는 과정

즉, 신경망은 단순히 값을 넣고 답을 내는 구조가 아니라, 오답을 줄이는 방향으로 내부 값을 계속 수정하면서 학습하는 구조입니다.
