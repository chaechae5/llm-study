# 퍼셉트론

## 퍼셉트론이란

퍼셉트론은 여러 입력을 받아 하나의 출력을 내보내는 가장 기본적인 인공신경망 모델입니다.
입력값에 가중치를 곱하고, 그 합을 바탕으로 최종 출력을 결정합니다.

즉, 퍼셉트론은 입력을 받아 단순한 계산을 수행한 뒤 결과를 출력하는 구조입니다.

## 기본 계산 방식

퍼셉트론의 기본 계산 흐름은 다음과 같습니다.

1. 입력값에 가중치를 곱합니다.
2. 각 결과를 모두 더합니다.
3. 기준값인 임계값과 비교합니다.
4. 최종 출력을 결정합니다.

정리하면, 여러 입력 중 어떤 값을 더 중요하게 볼지를 가중치로 조정하고, 그 결과를 바탕으로 판단하는 방식입니다.

## 퍼셉트론의 활용 예시

| 만들고 싶은 것 | 입력 | 출력 |
| --- | --- | --- |
| 스팸 메일 분류기 | 메일 내용 | 스팸 / 정상 |
| 고양이 사진 분류기 | 픽셀값 | 고양이 / 아님 |
| 집값 예측기 | 면적, 위치, 층수 | 가격 |
| 번역기 | 한국어 문장 | 영어 문장 |
| 대화형 AI | 질문 텍스트 | 답변 텍스트 |

이처럼 입력을 받아 특정한 결과를 만들어내는 구조라면, 넓은 의미에서 퍼셉트론 기반 사고방식으로 이해할 수 있습니다.

## 퍼셉트론의 한계

단일 퍼셉트론은 매우 단순한 문제만 해결할 수 있습니다.

특히 데이터를 직선 하나로 구분할 수 있는 경우에는 동작하지만, XOR와 같은 비선형 문제는 해결할 수 없습니다.
즉, 퍼셉트론 하나만으로는 복잡한 패턴을 학습하는 데 한계가 있습니다.

## 퍼셉트론에서 신경망으로의 확장

이 한계를 해결하기 위해 퍼셉트론을 여러 개 연결한 구조가 등장했습니다.

1. 퍼셉트론
2. 여러 퍼셉트론을 층으로 연결한 구조: 다층 퍼셉트론(MLP)
3. MLP를 포함하는 더 넓은 개념: 신경망(Neural Network)
4. 신경망을 깊게 학습시키는 방식: 딥러닝(Deep Learning)
5. 이를 대규모로 확장한 현대적 모델: GPT, Claude 등의 대규모 AI

즉, 오늘날의 AI 모델도 근본적으로는 퍼셉트론과 신경망 개념 위에서 발전한 구조입니다.

다만 GPT와 같은 최신 모델은 단순한 MLP만으로 이루어진 것이 아니라, Attention을 활용하는 Transformer 아키텍처를 기반으로 합니다.

## 퍼셉트론을 얼마나 쌓을 것인가

모델의 층 수와 노드 수는 사람이 설계합니다.
명확한 정답이 있는 영역은 아니며, 문제 특성, 기존 사례, 실험 결과를 바탕으로 결정합니다.

## 층별 의미

### 1. 입력층

입력 데이터의 형태를 받는 층입니다.
입력 개수는 데이터 형태에 따라 결정됩니다.

- `28 x 28` 픽셀 이미지 -> 입력 784개
- 키, 몸무게, 나이 -> 입력 3개
- 문장 입력 -> 토큰 단위 입력

### 2. 출력층

최종 결과를 내보내는 층입니다.
출력 개수는 수행하려는 작업에 따라 결정됩니다.

- 스팸 분류 -> 출력 1개
- 숫자 인식 -> 출력 10개
- 집값 예측 -> 출력 1개
- 기계 번역 -> 단어 또는 토큰 단위 출력

### 3. 은닉층

입력과 출력 사이에서 특징을 학습하는 층입니다.
가장 설계 여지가 큰 부분입니다.

은닉층을 설계할 때는 보통 두 가지를 구분해서 봅니다.

- 층 수 -> 모델의 깊이
- 노드 수 -> 각 층이 표현할 수 있는 정보량

일반적인 경향은 다음과 같습니다.

- 문제가 단순한 경우 -> 얕은 구조
- 문제가 복잡한 경우 -> 더 깊거나 넓은 구조
- 대규모 언어모델 -> 매우 많은 층과 파라미터 사용

## 실제 설계 방식

현실에서는 보통 다음 순서로 접근합니다.

1. 작은 구조로 시작합니다.
   - 우선 단순한 모델로 성능을 확인합니다.
2. 성능이 부족하면 확장합니다.
   - 층 수 또는 노드 수를 증가시킵니다.
3. 유사 문제의 논문, 오픈소스, 선행 구조를 참고합니다.
   - 검증된 구조를 우선 활용합니다.
4. 반복 실험을 진행합니다.
   - 여러 하이퍼파라미터 조합을 비교한 뒤 최적 구조를 선택합니다.

즉, 모델 구조 설계는 공식 암기보다는 경험적 최적화에 가까운 영역입니다.

## 핵심 트레이드오프

| 구조 크기 | 장점 | 단점 |
| --- | --- | --- |
| 너무 작은 경우 | 빠른 학습, 단순한 구조 | 복잡한 문제 표현 한계 |
| 너무 큰 경우 | 복잡한 패턴 학습 가능 | 학습 시간 증가, 과적합 위험 |

## 과적합(Overfitting)

과적합은 학습 데이터에는 잘 맞지만, 새로운 데이터에서는 성능이 떨어지는 현상입니다.

결국 목표는 문제를 해결할 수 있을 만큼 충분한 표현력을 가지면서도, 불필요하게 크지 않은 구조를 설계하는 것입니다.

## 정리

- 퍼셉트론은 입력값과 가중치를 이용해 출력을 결정하는 가장 기본적인 신경망 모델입니다.
- 단일 퍼셉트론은 단순한 문제만 해결할 수 있습니다.
- 복잡한 문제를 다루기 위해 여러 퍼셉트론을 층으로 연결한 다층 구조가 등장했습니다.
- 이 다층 구조가 신경망으로 발전했고, 더 깊고 큰 모델로 확장되면서 딥러닝과 GPT 같은 현대 AI 모델로 이어졌습니다.
- 즉, 현대 AI는 퍼셉트론 개념이 확장되고 고도화된 결과물로 이해할 수 있습니다.
